# marketplace-automation
    A simple Python automation tool to batch-process resumes and generate ATS-friendly resumes and cover letters.

## PROBLEM STATEMENT 
    Job seekers often need to tailor resumes and cover letters at scale for ATS systems. This tool automates that process.

## EXAMPLE WORKFLOW
    input_resumes/ → script → output_resumes/
